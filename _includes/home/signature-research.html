Thesis Arc — Calibrated Uncertainty, in Three Papers

Three papers on a single idea: sequential decision-making that is honest about its confidence. The setting in my Ph.D. was early sepsis prediction; the methods are portable. Each paper ships with open-source code.

OnAI-Comp

A pool of models competes. A trusted selector picks the right one per case.

Many models are better than one — but only if you can tell which one is right for this patient. OnAI-Comp does the telling, with an interpretable explanation for every choice.

Read paper View code

Sepsyn-OLCP

Every prediction ships with a calibrated confidence interval.

Point predictions don't cut it when the cost of being wrong is someone's life. Sepsyn-OLCP wraps the pipeline in split conformal prediction, so the system knows — and says — when it shouldn't be trusted.

Read paper View code

NeuroSep-CP-LCB

Learn a patient-specific reward model, safely, from offline data alone.

A neural network replaces the model pool — expressive enough to be personal — with a conformal lower-confidence-bound bonus that prevents the policy from wandering into territory the data never visited.
